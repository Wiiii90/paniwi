# Code of Conduct

This project should be a friendly place to collaborate.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent, especially in reviews and issue discussions.
- Keep feedback focused on the work, not the person.
- Be mindful that contributors may have different levels of experience.

## Unacceptable Behavior

Harassment, insults, discrimination, threats, or deliberately disruptive behavior are not welcome.

## Reporting

If you notice a problem, please contact the maintainer through the GitHub profile linked to this repository. Reports will be handled with care and discretion.

## Scope

This code of conduct applies to project spaces such as issues, pull requests, discussions, and related collaboration.
